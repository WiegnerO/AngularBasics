Interpolation


Within the interpolaion.component.ts file we have a attribute within the InterpolationComponent class called name and we assigned it the value "ori".
To get the value of this varibale within the temple we use double curly braces {{}} to specify the name property.
The syntax of an expression within double curly braces is known as interpolation
Interpolation works by asking angluar to evalute the content within the curly braces and display the value in the browser
Can output class values {{name}} : {{name}}
Can do aritmetic operations In between the brackets {{1 + 2}} : {{1 + 2}}
Can concatinate strings {{"welcome " + name}} : {{"welcome " + name}}
Can use Javascript properties and methods {{name.length}} : {{name.length}}
{{name.toUpperCase()}} :{{name.toUpperCase()}}
Can use methods defined in the component class {{greetUser()}} : {{greetUser()}}
Cannot do assignments with interpolation Will display [Parser Error: Bindings cannot contain assignments ...]
Cannot access global Javascript variables {{window.location.href}} : Will display [Property 'window' does not exist on type 'TestComponent ...]
To get around this you can create a property in the the InterpolationComponen class which links to the url location
public siteUrl = window.location.href
{{siteUrl}} : {{siteUrl}}
