How to make a custome pipe


  1. Generate the pipe
    • In the console type:
      ng g p < pipename >
  2. Add the pipe logic
    • In the <pipe-name>.pipe.ts file there is a transfrom attribute add the logic there
    • The pipe can have 0 -> n arguments
  3. Add the imports
      In the app.module.ts file make sure the pipe is in the imports and the declorations
  4. Use the pipe in the corresponding html file
    • To use the pipe you interpolate it and input the arguments
    • {{'<input-vale>' | <pipe-name> : <argument-value1> : ... : <argument-valueN>}}
