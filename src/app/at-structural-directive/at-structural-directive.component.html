Structural Directives

Structural directives are directives thta let you add or remove HTML elements from the dom
there are 3 main types
  1. ngIf
  2. ngSwitch
  3. ngFor

*ngIF

ngIf and Ng ngSwitch are used for conditional statments while ngFor is used to list lists of elements
< a *ngIf="true" > Hi < /a > : Hi
< a *ngIf="false" > Hi < /a > Hi: Here the ngIf value is false therefore Hi wont be visable
This element is removed from the DOM it is NOT the same as setting the css element to hide

Setting a value is useless therefore the component can have a boolean attribute to toogle it on and off
< a *ngIf="displayNameTrue" > Hi < /a > : Hi
< a *ngIf="displayNameFalse" > Hi < /a > Hi: Here the ngIf value is displayNameFalse = false therefore Hi wont be visable
< h2 *ngIf="displayNameTrue ; else elseBlock" > Hi < /h2 >
