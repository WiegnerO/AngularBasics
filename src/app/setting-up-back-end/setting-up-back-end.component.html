How to set up the backend

In the terminal type npm init
It will prompt you for intial set up options and then will create a package.json file

In the terminal type npm install express --save
This will install the express dependancies and put them in the package.json file

                        "name": "fireappbackend",

                    {{ '}' }}

In the terminal type npm install -g nodemon
This will detect if there are any changes to the server files and refresh the page automatically
In the terminal to use nodemon type nodemon (server file name) nodemon server.js

Create a server file in the directory (it can be named anything) server.js
First we need to bring in the express framework and storing it in a constant var express = require('express');
Secondly we initialize the express framework and save it into another constant. var app = express();
Then we save the port of the server as a constant value, the process.env.PORT is going to check your environment variables to see if you already have a PORT defined there if not it will use PORT 4000.const PORT = process.env.PORT || 4000
There is a built in isten method and it expects the port number that our is located. You can pass a callback function as a second argument
To start the server in the terminal type node (servername)node server.js 

                    server.js
                  
                    var express = require('express');
                    var app = express();
                    const PORT = process.env.PORT || 4000

                    app.get('/', (req,res)=>{{ '{' }}
                        res.send('hello');
                        {{ '}' }})

                    app.listen(PORT, ()=>{{ '{' }}
                        console.log('heyoo')
                    {{ '}' }});
